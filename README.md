# it-logger
### usefulle for managing logs and tasks in It-industry

1. React
2. Redux
3. JSON-Server (mock backend api)

### Usage
 Install dependencies:
 ```
 npm install
 ```
 Run server:
 ```
 npm run dev # JSON-Server (:5000) and React (:3000)
 npm run json-server # Run only JSON-Server (:5000)
 ```
